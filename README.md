## 🚀 Proyectos

### 📦 Sistema de Inventario
Sistema web full stack para la gestión de productos con autenticación JWT.
- **Backend:** Java + Spring Boot + PostgreSQL → desplegado en Railway
- **Frontend:** Angular + TypeScript → desplegado en Vercel
- **Funcionalidades:** CRUD de productos, login con JWT, buscador en tiempo real

🔗 [Ver proyecto](https://inventario-frontend-delta.vercel.app) | [Repositorio](https://github.com/ingdanielanzola/inventario-frontend)
